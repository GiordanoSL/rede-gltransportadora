# Projeto de Infraestrutura de Redes - GL Transportadora 🚚🌐

Este repositório contém o projeto de rede desenvolvido no **Cisco Packet Tracer** para a empresa fictícia **GL Transportadora**. O objetivo foi projetar uma infraestrutura escalável, segura e eficiente conectando uma Matriz (Curitiba-PR) a uma Filial (São Paulo-SP).

## 🚀 Tecnologias e Conceitos Utilizados
*   **VLSM (Variable Length Subnet Mask):** Segmentação lógica eficiente para otimização do endereçamento IP.
*   **Roteamento:** Configuração de gateway central e comunicação entre unidades.
*   **Servidores Dedicados:** Implementação de serviços de DHCP, DNS, HTTP, FTP e SMTP na Matriz.
*   **Fibra Óptica:** Enlaces de alta velocidade (1000Mbps) para os backbones da rede.
*   **Wi-Fi:** Pontos de acesso para mobilidade operacional (vistorias e dispositivos móveis).

## 🏗️ Estrutura da Rede

### **Matriz: Curitiba**
*   **Setores:** TI (Servidores), RH, Financeiro, Gestão de Frotas e Monitoramento.
*   **Destaque:** Centralização dos serviços de rede para redução de custos operacionais e facilidade de gerenciamento.

### **Filial: São Paulo**
*   **Setores:** Operacional, Comercial e Marketing.
*   **Destaque:** Foco em mobilidade para vistorias de campo e integração total com os sistemas da Matriz.
